Решение линейных
уравнений

Главный принцип: при переносе любого элемента через знак равенства его знак меняется на противоположный — будь то число или переменная.

Основная идея

Принцип равновесия

Уравнение — это весы. Левая часть (ЛЧ) всегда должна быть равна правой части (ПЧ). Любое действие, выполненное с одной стороной, необходимо выполнить и с другой.

\( \text{ЛЧ} = \text{ПЧ} \quad \Rightarrow \quad \text{ЛЧ} \pm k = \text{ПЧ} \pm k \)
➕
Прибавить одинаковое число к обеим частям — баланс сохраняется
➖
Вычесть одинаковое число из обеих частей — баланс сохраняется
✖️
Умножить обе части на одинаковое число — баланс сохраняется
➗
Разделить обе части на одинаковое число — баланс сохраняется
Ключевое правило
⚡ Правило переноса (смена знака)

Когда мы переносим любой элемент через знак равенства — будь то число или переменная, перенос вправо или влево — мы обязательно меняем его знак на противоположный.

+ 5 → переходит через = → − 5
− 3x → переходит через = → + 3x

Алгоритм решения

Пошаговый метод

01

Раскрыть все скобки

Используй закон распределения — умножь каждый элемент в скобках на то, что стоит перед ними.

\(a(b+c) = ab + ac\)
02

Перенести все элементы с переменной ВЛЕВО

Перенеси все слагаемые, содержащие x, в левую часть. При переходе через = — меняй знак на противоположный.

+3x справа → −3x слева
03

Перенести все числа ВПРАВО

Перенеси все числа (без x) в правую часть. При переходе через = — меняй знак на противоположный.

+7 слева → −7 справа
04

Привести подобные слагаемые

Сложи все x-слагаемые слева, сложи все числа справа.

05

Найти x — разделить обе части

Раздели обе части на коэффициент при x, чтобы найти решение.

\(4x = 20 \Rightarrow x = 5\)
06

Проверить ответ

Подставь найденное x в исходное уравнение — обе части должны быть равны.

Краткая справка
Ситуация Что делать Правило знака
\(+k\) в левой части Перенести вправо становится −k
\(-k\) в левой части Перенести вправо становится +k
\(+bx\) в правой части Перенести влево становится −bx
\(-bx\) в правой части Перенести влево становится +bx
Дробь \(\dfrac{ax+b}{c} = k\) Сначала умножить обе части на c знак не меняется
Скобки \(a(bx+c)\) Сначала раскрыть скобки знак пока не меняется
Разобранные примеры
Пример 1 · Базовый — переменные с обеих сторон
2x − 1 = 5 + x
① \(2x - 1 = 5 + x\) исходное уравнение
② \(2x - x = 5 + 1\) x переносим влево; −1 вправо → +1
③ \(x = 6\) приводим подобные
✓ x = 6 Проверка: 2(6)−1 = 11 = 5+6 ✔
Пример 2 · Скобки с обеих сторон
3(x − 6) = 2(2 − x) + 3
① \(3(x-6) = 2(2-x)+3\) исходное уравнение
② \(3x - 18 = 4 - 2x + 3\) раскрываем скобки
③ \(3x + 2x = 4 + 3 + 18\) −2x → +2x (влево); −18 → +18 (вправо)
④ \(5x = 25\) приводим подобные
⑤ \(x = 25 \div 5\) делим обе части на 5
✓ x = 5 Проверка: 3(−1)=−3 = 2(−3)+3 ✔
Пример 3 · Уравнение с дробью
(1 − 3x) / 2 = −4
① \(\dfrac{1-3x}{2} = -4\) исходное уравнение
② \(1 - 3x = -4 \times 2\) умножаем обе части на 2
③ \(1 - 3x = -8\) упрощаем правую часть
④ \(-3x = -8 - 1\) +1 переносим вправо → −1
⑤ \(-3x = -9\) упрощаем
⑥ \(x = (-9) \div (-3)\) делим обе части на −3
✓ x = 3 Проверка: (1−9)/2 = −4 ✔
Пример 4 · Двойные скобки + приведение
4(2x + 3) − 10 = 5(3 − x)
① \(4(2x+3)-10=5(3-x)\) исходное уравнение
② \(8x + 12 - 10 = 15 - 5x\) раскрываем обе скобки
③ \(8x + 2 = 15 - 5x\) приводим в левой части
④ \(8x + 5x = 15 - 2\) −5x → +5x (влево); +2 → −2 (вправо)
⑤ \(13x = 13\) упрощаем
✓ x = 1 Проверка: 4(5)−10=10=5(2) ✔
Пример 5 · Отрицательный знаменатель
(6 + 5x) / −2 = 7
① \(\dfrac{6+5x}{-2} = 7\) исходное уравнение
② \(6 + 5x = 7 \times (-2)\) умножаем обе части на −2
③ \(6 + 5x = -14\) упрощаем правую часть
④ \(5x = -14 - 6\) +6 переносим вправо → −6
⑤ \(5x = -20\) упрощаем
✓ x = −4 Проверка: (6−20)/(−2) = 7 ✔
